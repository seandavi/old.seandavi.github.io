01 Sep 2017

tensorflow / machine learning / deep learning / cloud / Data Science

What is TensorFlow? TensorFlow is an open source library for machine learning, developed by researchers and engineers at Google. TensorFlow is designed to run on multiple computers to distribute the training workloads, and Google’s Cloud Machine Learning Engine provides a managed service where you can run TensorFlow code in a distributed manner by using service APIs. TensorFlow often serves as a backend for other machine learning frameworks such as keras.
