Introduction

Analysis and comprehension of high-throughput biological data

  • Started in 2002
  • Builds on R, which includes 14,376 “contributed” software packages and more than 2 million users worldwide

Bioconductor is well-used and well-respected

  • More than 50,000 unique downloads per month
  • Approximately 21,700 PubmedCentral citations

But what is Bioconductor?

Bioconductor is a Toolkit for high-throughput biological data science built with reproducible computational research as a core value.

Bioconductor software supports all aspects the biological data science workflow.

  • Data import and export
  • Data management and accessibility
  • Data integration, including with public data
  • Analysis and statistics
  • High-level visualization
  • Efficiency and reproducibility, even at scale

Example 1: Build a PCA plot from Public Data

Filter gene expression by variance to find most informative genes.

sds = apply(exprs(gse),1,sd)
dat = exprs(gse)[order(sds,decreasing = TRUE)[1:500],]

Perform multidimensional scaling and prepare for plotting.

mdsvals = cmdscale(dist(t(dat)))
mdsvals = as.data.frame(mdsvals)
mdsvals$Type=factor(pData(gse)[,'cancer type:ch1'])
mdsvals$Normal = factor(pData(gse)[,'normal:ch1'])

And do the plot.

library(ggplot2)
ggplot(mdsvals, aes(x=V1,y=V2,shape=Normal,color=Type)) + 
    geom_point(size=4, alpha=0.6) + theme(text=element_text(size = 18))

Example 2: Melanoma Variants from NCI Genomic Data Commons Data

Use the GenomicDataCommons package to find and download variants from the TCGA cutaneous melanoma dataset.

library(GenomicDataCommons)
fnames = files() %>% 
    GenomicDataCommons::filter(~ cases.project.project_id=='TCGA-SKCM' & 
        data_type=='Masked Somatic Mutation' &
        data_format=='MAF' &
        analysis.workflow_type=='MuTect2 Variant Aggregation and Masking') %>%
    ids() %>%
    gdcdata()

And now take those data directly to maftools for analysis and visualization.

library(maftools)
melanoma = read.maf(maf = fnames[1])

Data access and management

Hubs and data packages give users access to curated and versioned public data.

  • AnnotationHub
    • Reference genomes
    • Gene models
    • ENCODE data
    • UCSC Tracks
  • ExperimentHub
    • Curated and versioned public datasets (TCGA, 1000Genomes)
    • Searchable metadata
  • Experiment data packages
    • Easy sharing of data within lab, institute, or internationally

Reproducible research

  • Everything in bioconductor is version controlled and versioned.
  • Package build system guarantees compatibility between packages.
  • Documentation and workflows are “literate”, so if they do not work or do not match code, Bioc build system alerts developer.
  • Strong reliance on interoperability to reduce “reinvention” (and decrease mistakes)
